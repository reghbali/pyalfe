pyalfe#

configure#

Configure the pipeline through a series of prompts.

Returns#

None

pyalfe configure [OPTIONS]

download#

pyalfe download [OPTIONS] [ASSETS]...

Arguments

ASSETS#

Optional argument(s)

run#

Runs the pipeline for an accession number.

Parameters#

accessionstr

the accession number for which you want to run the pipeline.

configstr, default: ~/.config/pyalfe/config.ini

the path to the config file.

input_dirstr

the path to the directory containing input input images

outpu_dirstr

the path to the directory containing output output images

modalitiesstr

comma separated modalities

targetsstr

comma separated target modalities

dominant_tissuestr, default=’white_matter’

dominant tissue

image_processorstr, default=c3d

image processor that is used by the pipeline.

image_registrationstr, default=greedy

image registration that is used by the pipeline.

overwritebool

if True, the pipeline overwrites existing output images.

Returns#

None

pyalfe run [OPTIONS] ACCESSION

Options

-c, --config <config>#
-m, --modalities <modalities>#
-t, --targets <targets>#
-id, --input-dir <input_dir>#

-dt, --dominant_tissue <dominant_tissue>#
Options:

white_matter | gray_matter | auto

-ow, --overwrite, -now, --no-overwrite#
-ip, --image-processor <image_processor>#
Options:

-ir, --image-registration <image_registration>#
Options:
